# 🎧 Spotify Music Data Analysis

# 📌 Sobre o Projeto

Este projeto tem como objetivo explorar e analisar dados de músicas do Spotify, buscando identificar padrões de popularidade, comportamento dos gêneros musicais e relações entre características sonoras das faixas.

A análise foi realizada utilizando Python para preparação dos dados e Power BI para construção de dashboards interativos.

🎯 Objetivos

Analisar a distribuição de popularidade das músicas
Investigar a relação entre características musicais e popularidade
Identificar padrões entre gêneros musicais
Explorar tendências ao longo do tempo

# 🛠️ Tecnologias Utilizadas
Python (Pandas, SQLite)
SQL
Power BI
Git & GitHub

# ⚙️ Etapas do Projeto
🔹 1. Coleta e Preparação dos Dados

Importação de arquivos CSV

Limpeza e padronização dos dados

Tratamento de duplicidades

Análise exploratória inicial (EDA)


🔹 2. Modelagem e Análise

Criação de queries SQL para análise

Agregações por artista, gênero e ano

Cálculo de métricas como média de popularidade


🔹 3. Visualização (Power BI)

O dashboard foi desenvolvido com foco em clareza e storytelling, contendo:

Relação entre acousticness e popularidade (scatter plot)

Popularidade média ao longo dos anos

Média de popularidade por artista

Gêneros com maior popularidade média

Indicadores principais (cards)


# 📊 Principais Insights

A popularidade das músicas não depende de uma única variável, mas sim de múltiplos fatores combinados
Não foi identificada uma correlação direta entre acousticness e popularidade
Alguns gêneros apresentam maior consistência em popularidade média
A análise sugere que fatores além das características sonoras influenciam o sucesso das músicas
